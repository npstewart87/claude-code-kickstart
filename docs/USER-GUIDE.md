# Your Claude Hub — How To Use It

Everything runs on your own private server in the cloud. Your laptop is just
the window into it. Three things to know, that's it.

## One-time setup (your admin will help, ~10 minutes)

1. **Install Tailscale** (from tailscale.com/download — Windows). your admin logs it
   in. After that it just sits in your tray saying "Connected." Never touch it.
2. **Install Windows Terminal** (Microsoft Store, free, probably already there).
3. **Install Obsidian** (obsidian.md, free) and **Syncthing** (syncthing.net) —
   this gives you the pretty notes app. your admin wires them up once.

## Daily use — the only thing to remember

1. Open **Windows Terminal**
2. Type: `ssh <user>@<your-hub>` and press Enter
3. You'll see **CLAUDE HUB**. Type `go` and press Enter.
4. Talk to it like a person. That's the whole thing.

It will list your projects when you start, keep everything organized for you,
and save notes automatically. You never need to manage files or folders —
just tell it what you want.

### Things you can say

- "Show my projects"
- "Let's keep working on <project>"
- "I have an idea — I want to build ..."
- "Research <anything> for me"
- "What did we do last time?"

## Your notes (Obsidian)

Open Obsidian on your laptop — your "second brain." Claude fills it in as you
work: every project gets a note, every idea gets saved. The graph view (the
cool web of dots) grows by itself.

## If something seems broken

Tell whoever set this up for you — they can fix anything remotely.
Don't try to fix the server yourself — you can't break anything by chatting,
so chat away.
